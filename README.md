## Installation

```
# backup your config
mv ~/.config/nvim{,.bak}

# clone this config
git clone https://github.com/steffenbew/lazyvim-config ~/.config/nvim

# install dependencies
brew install nvim fd ripgrep lazygit

# start neovim!
nvim
```
